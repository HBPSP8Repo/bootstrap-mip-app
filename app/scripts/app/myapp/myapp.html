Hello HBP !

Code your app

You can create your app view in this file (myapps.html) !
You can add some CSS files in the app/style/css/myapp/ folder and include them in the app/index.html file (but don't break our graphic charter) !
You can add some Javascript files in the app/scripts/app/myapp/ folder and include them in the app/index.html file !
You can add some images in the app/images/myapp/ folder !
You can use AngularJS (out of the box) ! Your controller is app/scripts/app/myapp/myapp.controller.js !

Use Grunt and Bower

You can serve your app with the command `grunt serve` !
You can build your app with the command `grunt build` !
You can use Grunt and/or Bower to install things (but you have to tell us) !
